# dns-filter-lists

A collection of DNS filter lists for use with DNS resolvers like AdGuard, AdGuard Home, etc.

## Usage

To use these filter lists, you can add them to your DNS resolver's configuration. For example, in AdGuard Home, you can go to the "Filters" section and add the URLs of the lists you want to use.

## Available Lists

The following lists are for my personal use and may not be suitable for everyone:

- Allowlist: `https://raw.githubusercontent.com/tillstuder/dns-filter-lists/main/allowlist.txt`
- Blocklist: `https://raw.githubusercontent.com/tillstuder/dns-filter-lists/main/blocklist.txt`
